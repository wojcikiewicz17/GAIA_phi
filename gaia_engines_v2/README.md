# gaia_engines_v2/ — Engines RAF v2

## Propósito
Módulo dedicado à execução de engines simbólicas/analíticas, com API enxuta e implementação centralizada.

## Estrutura de arquivos
- `include/raf_engine.h`: contrato da engine.
- `src/raf_engine.c`: implementação do ciclo de vida e operação.

## Conceitos principais
- **Engine como unidade de execução** com inicialização, execução e finalização.
- **Métricas de execução** para inspeção de comportamento.
